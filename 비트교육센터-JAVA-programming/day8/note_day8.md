## GUI 라이브러리

### awt
GUI 껍데기

기존의 os의 도움을 받아서 만듬

### swing
JVM 에서 돌아간다
awt가 os의 도움을 받아야해서 느리고 os에 따라 모양이 다른다

그래서 추가적으로 라이브러리가 만들어졌다.
